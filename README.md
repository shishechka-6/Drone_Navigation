# Алгоритм детекции окон пролётов для дрона

## Описание проекта

Проект представляет собой программную реализацию алгоритма **детекции пролётных окон** на двумерной карте.  
Алгоритм получает, анализирует свободные области и выделяет фрагменты, через которые может пройти объект (например, дрон или робот).

Основная задача:  
**автоматически выявлять проходы достаточной ширины и формировать список "окон пролётов", доступных для безопасного манёвра.**

Алгоритм может использоваться в навигационных системах, симуляторах и в системах локального планирования.

---

## Основные возможности

- поиск свободных областей, соответствующих требованиям ширины;
- выделение потенциальных пролётных окон;
- фильтрация ложных срабатываний;
- вычисление координат и границ проходов;
- формирование списка кандидатов для навигационного модуля.

---

## Принцип работы алгоритма

1. Получение карты.
2. Нормализация и бинаризация по порогу.
3. Поиск свободных сегментов в направлении движения.
4. Определение минимальной требуемой ширины.
5. Объединение соседних свободных областей.
6. Формирование структур:
   - координаты окна,
   - ширина,
   - положение центра,
   - пригодность для пролёта.
7. Возврат списка найденных окон.

---

## Стек проекта

- Python
- NumPy
- OpenCV
- ROS 2 для интеграции
- Gazebo / симуляция

---


## Применение

- определение проходов между стенами;
- выбор безопасной зоны для манёвра;
- локальная навигация;
- фильтрация траекторий;
- симуляции с виртуальными роботами/дронами.

---

## Статус проекта
